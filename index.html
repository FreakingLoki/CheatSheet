Basic HTML Tags
These are some of the basic HTML elements and tags that are used to make up the majority of a webpage. Follow the links for more in-depth information and examples.

<!DOCTYPE html> self closing Doctype Declaration Tells the browser that the document being viewed is an HTML document. Every HTML document must include this on line 1. Further Reading and Examples
<html> </html> HTML Encloses the entirety of the HTML document excluding only the Doctype declaration. Further Reading and Examples
<head> </head> Head Contains metadata such as the title of the page and links for connecting CSS styling or fonts. Information in this section of the page will not be displayed on the page. Further Reading and Examples
<body> </body> Body Contains the parts of the page that are to be displayed within the browser. Usually divided up into sections such as header, main, footer, etc. See Semantic HTML section for more information. Further Reading and Examples
<h1> </h1> Header Used to add header-level text to a page. Has multiple levels ranging from largest (h1) to smallest (h6). Further Reading and Examples
<p> </p> Paragraph Used to add paragraphs of text to a page. Further Reading and Examples
<!-- --> Comment Use this important element to add comments to HTML code so that anyone reveiwing the code can understand what it is trying to accomplish. Comments can enclose HTML code to prevent that code from being run Further Reading and Examples
<img> self closing Image Used to add images to a page. Can be used to embed external images from another page or internal images from the website's files Further Reading and Examples
<video> </video> Video Adds a video to a page. Be sure to include the controls attribute to be user friendly. Further Reading and Examples
<audio> </audio> Audio Adds an audio element (music, speech, or other) to the page. Be sure to include the controls attribute to be user friendly. Further Reading and Examples
<a> </a> Anchor Used to add links to other pages, files, or even email addresses. Can also link within the page it is used on. Further Reading and Examples
<br> self closing Line Break Adds extra blank lines in bewteen elements. Usually it is better to adjust spacing with CSS properties. Further Reading and Examples
<em> </em> Emphasis Used to signify emphasis on a section of text. Further Reading and Examples

These elements are used to contruct HTML Tables. These are powerful formatting tools and can be used to build entire pages such as this one.
<table> </table> Table Encloses the entire table including the head, foot and body. Further Reading and Examples
<thead> </thead> Table Head This element is used to make the header of a table, usually seen as names or descriptions of the rows or columns. Further Reading and Examples
<tbody> </tbody> Table Body Used to enclose the main data rows of the table. Further Reading and Examples
<tfoot> </tfoot> Table Foot This element encloses the footer of the table which is usually used for summary info or citations. Further Reading and Examples
<tr> </tr> Table Row This element adds a new row to a table. Further Reading and Examples
<th> </th> Table Header This element is used to denote a cell as a header of a group of cells Further Reading and Examples
<td> </td> Table Data Cell This element adds the individual cells within a Table Row. Further Reading and Examples

These tags are used to make your HTML code easier to read and understand. Combine these tags with good comments and formatting to make your code simple to read.
<section> </section> Section Denotes a section within the webpage. Should always include a heading. Further Reading and Examples
<article> </article> This tag is meant to denote an article such as a forum post or newspaper article Additional tags may be used within this one to format the article Further Reading and Examples
<aside> </aside> Used to enclose a portion of the page that is indirectly related to the main content. Further Reading and Examples
<address> </address> Used to describe the section of a page that includes addresses or other forms of contact information. Further Reading and Examples
